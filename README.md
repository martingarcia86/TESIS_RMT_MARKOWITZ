## Descripción

Este trabajo estudia el problema de la estimación de matrices de covarianza en alta dimensión mediante herramientas de la Teoría de Matrices Aleatorias (Random Matrix Theory, RMT) y su aplicación en la construcción de portafolios de mínima varianza.

Se desarrollan los fundamentos teóricos relacionados con:

- Distribución de Marčenko-Pastur.
- Matrices de Wishart.
- Ruido espectral y separación señal-ruido.
- Métodos de limpieza espectral.
- Estimadores de covarianza basados en RMT.
- Optimización de portafolios de Markowitz.

Además, se presenta una aplicación empírica utilizando datos históricos del mercado accionario mexicano.
